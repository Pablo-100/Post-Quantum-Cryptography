Chapitre 01 — Contexte

Bits, Moore's Law
& l'Informatique Quantique

La 4ème génération des ordinateurs utilise des transistors et des bits. Gordon Moore (Intel, 1965) a observé que le nombre de transistors double tous les 2 ans. Cette loi atteint ses limites physiques à l'échelle atomique.

Moore's Law — Formule
2^((Année - 1949) / 2) = Bits └─ (Année-1949) doit être pair Ex: 2025 → 2^((2025-1949)/2) = 2^38 ≈ 274 milliards
⚠ Limite Physique

Transistors à 1–2 nm = quelques atomes. L'effet tunnel traverse les barrières quantiquement. La miniaturisation classique est terminée.

Qubits &
Mathématiques Quantiques

Notation de Dirac, kets, bras, produit interne, normalisation, sphère de Bloch, registres n-qubits, produit tensoriel et entanglement.

Ket |ψ⟩ — Vecteur colonne
|ψ⟩ = [α₀] αᵢ ∈ ℂ [α₁] [α₂] Base computationnelle: |0⟩ = [1] |1⟩ = [0] [0] [1]
Note

Les kets peuvent avoir n'importe quelle dimension. Les kets unitaires (norme=1) sont des états quantiques valides.

Bra ⟨ψ| — Conjugué
⟨ψ| = [α₀*, α₁*, α₂*] └─ conjugués complexes du ket
Produit Interne ⟨φ|ψ⟩

Entanglement Quantique (Intrication)

Un état 2-qubits est entanglé s'il ne peut pas être décomposé en produit tensoriel:

Mesurer q₀ détermine instantanément q₁ — "spooky action at a distance" (Einstein)

États de Bell (base orthonormée)
|Φ⁺⟩ = (|00⟩ + |11⟩)/√2
|Φ⁻⟩ = (|00⟩ - |11⟩)/√2
|Ψ⁺⟩ = (|01⟩ + |10⟩)/√2
|Ψ⁻⟩ = (|01⟩ - |10⟩)/√2
Test d'entanglement
Est-ce que |Φ⁺⟩ factorisable? (|0⟩+|1⟩)⊗(|0⟩+|1⟩) = ½(|00⟩+|01⟩+|10⟩+|11⟩) ≠ (|00⟩+|11⟩)/√2 ← ENTANGLÉ! ✓

Chapitre 03 — Circuits

Quantum Gates
& Circuits

Transformations unitaires (U†U=I) — réversibles et conservant les normes. Nombre inputs = outputs.

Créer un état de Bell
H⊗I sur |00⟩ → (1/√2)(|0⟩+|1⟩)⊗|0⟩ CNOT → (1/√2)(|00⟩+|11⟩) = |Φ⁺⟩ ✓
Toffoli Gate (CCNOT) — 3 Qubits

Flip le target si deux controls = 1. Gate universelle pour calcul classique réversible.

|x,y,z⟩ → |x,y,z⊕(x·y)⟩ |x=0,y=?,z⟩ → |x,y,z⟩ (rien si x=0) |x=1,y=0,z⟩ → |x,y,z⟩ (rien si y=0) |x=1,y=1,z⟩ → |x,y,z̄⟩ ← flip si x=y=1 U_Toffoli = block_diag(I₂,I₂,I₂,X₂) [8×8]

Multi-Controlled X gates généralisent CNOT/Toffoli. Essentielles pour MODEXP dans Shor's algorithm.

Chapitre 04 — La Menace

Shor's Algorithm
Brise RSA

RSA repose sur l'impossibilité de factoriser de grands N. Shor (1994): factorisation en temps polynomial O((log N)³) — exponentiellement plus rapide que tout algo classique.

RSA — Comment ça marche
1
Choisir p,q premiers

Deux grands nombres premiers (512 bits)

2
N = p×q

Modulus public (2048 bits)

3
φ(N) = (p-1)(q-1)

Fonction d'Euler

4
e coprime à φ(N)

Exposant public

5
d: e·d ≡ 1 mod φ(N)

Clé privée

Chiffrement: C = Mᵉ mod N Déchiffrement: M = Cᵈ mod N
CLASSIQUE (GNFS)

Solutions Post-Quantiques

PQC (Post-Quantum Crypto): Solutions classiques résistantes aux QC. Basées sur lattices, hash, codes. Remplacent RSA/ECC sans matériel quantique. Standards NIST 2024: FIPS 203, 204, 205.

QKD (Quantum Key Distribution): Utilise des systèmes physiques quantiques. BB84 (IBM, 1984) — 1er protocole QKD. QRNG + propriétés quantiques = sécurité physique.

Chapitres 07 & 08 — CRYSTALS-Kyber

CRYSTALS-Kyber
ML-KEM, FIPS 203

Standard KEM basé sur MLWE (Module Learning With Errors) sur réseaux de modules. Résistant aux ordinateurs quantiques.

Qu'est-ce qu'un Lattice?

Ensemble de points formés par combinaisons entières de vecteurs bases:

L(B) = {Σ aᵢbᵢ : aᵢ ∈ ℤ}
SVP — Shortest Vector Problem

Trouver le vecteur le plus court ≠ 0. NP-hard même pour QC!

CVP — Closest Vector Problem

Trouver le point le plus proche d'une cible. Base de la sécurité Kyber.

Lattice 3D Rotatif — CVP

Bleus=lattice · Rouge=target CVP · Jaune=solution

LWE Sans Bruit — Facile
A×s = t — résolvable! → Élimination gaussienne
LWE Avec Bruit — Impossible!
A×s + e = t (mod q) e = petit bruit aléatoire → NP-hard même pour QC!
MLWE — Version Polynomiale
Anneau: Zq[X]/(Xⁿ+1) q=3329, n=256 Réduction: Xⁿ = -1 A(X) ∈ M^(k×k) polynômes s(X) petits coefficients ← secret e(X) petit bruit ← erreur t(X) = A(X)·s(X) + e(X) ← clé pub

Pourquoi q=3329?

3329 est premier et 3329 ≡ 1 (mod 256) — permet le NTT (Number Theoretic Transform) pour multiplier les polynômes en O(n log n).

Protocole KEM
A
Alice — KeyGen
Génère A, s, e
t = A×s + e    pk=(A,t), sk=s
↓
Alice publie pk=(A,t)
s reste secret
B
Bob — Encapsulation
Encode m, tire r, e₁, e₂
u=Aᵀr+e₁   v=tᵀr+e₂+enc(m)
↓
Bob envoie c=(u,v)
A
Alice — Decapsulation
m' = decode(v - sᵀ·u) = m ✓
Correctness Proof
v - sᵀ·u = (tᵀr+e₂+enc(m)) - sᵀ(Aᵀr+e₁) = sᵀAᵀr + eᵀr + e₂ + enc(m) - sᵀAᵀr - sᵀe₁ = enc(m) + [petit bruit] Si bruit petit → décodage correct!
Encode / Decode
encode(0) = 0 encode(1) = ⌊q/2⌋ = 1664 decode: xᵢ proche 0 → bit=0 xᵢ proche q/2 → bit=1
Exemple Scalaire (k=2, n=1, q=17)
A=[[6,5],[3,2]] s=[1,2]ᵀ e=[1,1]ᵀ As = [6+10, 3+4]ᵀ = [16,7]ᵀ t = As+e = [17,8]ᵀ = [0,8]ᵀ (mod 17) pk = (A, [0,8]ᵀ) sk = s = [1,2]ᵀ ← SECRET!
m=1, r=[2,1]ᵀ, e₁=[1,0]ᵀ, e₂=1 encode(1) = ⌊17/2⌋ = 8 u = Aᵀr+e₁ = [15,12]ᵀ+[1,0]ᵀ = [16,12]ᵀ v = tᵀr+e₂+8 = 8+1+8 = 17 = 0 (mod 17) c = (u,v) = ([16,12]ᵀ, 0)
sᵀu = [1,2]×[16,12]ᵀ = 40 40 mod 17 = 6 v - sᵀu = 0-6 = -6 = 11 (mod 17) dist(11,0)=11 dist(11,8)=3 ← proche q/2! → bit = 1 ✓
Exemple Polynomial (k=2, n=4, q=17)
Anneau: Z₁₇[X]/(X⁴+1) Règle: X⁴=-1, X⁵=-X, X⁶=-X², ... s = [-X³-X²+X, -X³-X]ᵀ ← petits e = [X², X²-X]ᵀ ← bruit m=11=(1011)₂ → m_b = X³+X+1 encode(m) = 9X³+9X+9

Résumé Mathématique
Sécurité

Basée sur MLWE — trouver s dans A×s+e=t est NP-hard même pour QC sur lattices de modules.

Efficacité NTT

Number Theoretic Transform — multiplication polynomiale en O(n log n) au lieu de O(n²).

Pourquoi Xⁿ+1?

n=256 → factorisation NTT optimale, arithmétique efficace et sécurité maximale.

Paramètre η

Coefficients de s,e tirés d'une distribution binomiale centrée Bη — petits mais sécurisés.
